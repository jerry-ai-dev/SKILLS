# 视频推荐指南

对于前置知识表中标注了 🎬 的概念，或学生表达"不太理解"、"能不能更直观一点"时，老师应推荐视频辅助理解。

## 视频查找规则

所有推荐视频统一存储在 [video_list.md](video_list.md) 中。推荐流程：

1. 根据当前讲解的概念，找到对应的**关键词**
2. 在 `video_list.md` 中按关键词匹配，找到对应视频
3. 如果该关键词对应的视频标题和链接为空，则**不推荐视频**（跳过视频推荐环节）
4. 如果有视频信息，按下方格式推荐给学生

## 推荐格式

为学生推荐视频时，使用以下格式：

```
📺 推荐视频：
- 概念：[概念名称]
- 视频：[视频标题]
- 链接：[视频链接]
- 备注：[备注信息，如有]
```

如果同一概念在 `video_list.md` 中有多条记录，可以全部列出供学生选择。

## 推荐时机

- **Step 2**：讲解学生选中的前置概念时
- **Step 3**：课内遇到特别抽象的知识点时
- **Step 9**：延伸学习资源环节
- **任何时候**：学生主动表示难以理解时
